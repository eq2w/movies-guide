# Проект: онлайн платформа  «VK Маруся»🎥

🔗 Установка:

Установка зависимостей:
```sh 
npm install
```
Запуск сервера разработки:
```sh
npm run dev
```
Запуск production сборки:
```sh
npm run build
```

Preview:
```sh
npm run preview
```

🛠️Stack:
- React		
- React Router
- React Query
- React Redux
- React Hook Form
- Zod
- TypeScript
- Sass
- Vite
